# 复用成熟度评估问卷

> **版本**: 2026-06-06
> **对齐标准**: ISO/IEC 26566:2026, RiSE Maturity Model, RCMM, NASA RRL
> **用途**: 组织级复用能力诊断与改进路径规划

---

## 评估说明

- **评估维度**: 过程 (Process) / 技术 (Technology) / 组织 (Organization) / 管理 (Management)
- **评分规则**: 每题 1-5 分，1=完全未做到，5=成熟优化
- **级别判定**:
  - L1 初始: 总分 < 80
  - L2 管理: 80-119
  - L3 定义: 120-159
  - L4 量化: 160-199
  - L5 优化: ≥ 200
- **问卷题数**: 50 题，每维度 12-13 题

## 评分与解读

### 分数计算

```
总分 = Σ(P01..P13) + Σ(T01..T12) + Σ(O01..O13) + Σ(M01..M12)
     ∈ [50, 250]
```

### 成熟度判定

| 总分范围 | 级别 | 特征 | 改进建议 |
|---------|------|------|---------|
| 50-79 | L1 初始 | 临时复用，克隆为主 | 建立基础资产管理，定义复用角色 |
| 80-119 | L2 管理 | 项目级复用，有管理 | 建立组织级标准，引入 IDP |
| 120-159 | L3 定义 | 组织级复用，标准化 | 建立度量体系，引入 FinOps |
| 160-199 | L4 量化 | 可度量复用，优化 | 引入 AI 辅助，自动化治理 |
| 200-250 | L5 优化 | 持续改进，自动复用 | 行业输出，生态扩展 |

### 维度雷达图建议

将四个维度得分绘制成雷达图，识别短板：

- 过程分低 → 流程未标准化
- 技术分低 → 工具链缺失
- 组织分低 → 文化/激励不足
- 管理分低 → 战略/度量缺失
