# 文档模板

> **版本**: 2026-06-06
> **定位**: 为知识库提供统一的文档结构模板

---

## 使用方式

创建新文档时，复制本模板并替换占位符内容。

---

```markdown
# [标题]

> **版本**: YYYY-MM-DD
> **对齐标准**: [相关标准]
> **定位**: [一句话说明本文档在知识体系中的位置]

---

## 目录

1. [第一节](#1-第一节)
2. [第二节](#2-第二节)
3. [第三节](#3-第三节)

---

## 1. 第一节

### 1.1 子节

内容...

### 1.2 子节

内容...

---

## 2. 第二节

> **公理 X.Y** (公理名称): 公理内容...

> **定理 X.TZ** (定理名称): 定理内容...

---

## 3. 第三节

### 检查清单

- [ ] 检查项 1
- [ ] 检查项 2
- [ ] 检查项 3

---

## 参考

- [链接 1]
- [链接 2]

---

> 最后更新: YYYY-MM-DD
```

---

## 文档编写规范

1. **标题层级**: 使用 `#` 到 `####`，避免过深嵌套
2. **代码块**: 使用 fenced code blocks，标注语言
3. **表格**: 使用 Markdown 表格呈现对比信息
4. **强调**: 使用 `**粗体**` 强调关键概念
5. **引用**: 公理和定理使用 `>` 块引用格式
6. **最后更新**: 每个文档末尾必须包含最后更新日期
