# 交互式复用决策工具

> **定位**: 支持六阶段复用决策流程的交互式工具（Web/CLI）
> **计划阶段**: Phase 6（2027-Q4）
> **技术栈**: Python + Streamlit（按决策 3A）

---

## 六阶段复用决策流程

1. **识别** — 发现潜在可复用资产
2. **评估** — 质量、成熟度、合规性评估
3. **适配** — 计算 AAF、修改范围
4. **集成** — 架构兼容性、依赖影响
5. **验证** — 测试、形式化验证、SBOM 审查
6. **治理** — 度量、成熟度、成本分摊

---

## 计划功能

- 上传/输入资产元数据，自动生成复用建议
- 集成 `cocomo-calculator.py` 计算工作量
- 集成 `maturity-assessment-cli.py` 评估成熟度
- 生成复用决策报告（PDF/Markdown）
